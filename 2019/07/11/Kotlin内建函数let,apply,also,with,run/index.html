Kotlin内建函数let,apply,also,with,run
发表于 2019-07-11| 更新于 2020-07-07
阅读量:

let

定义

  • 被调用对象通过参数传入
  • 返回类型是lambda表达式返回的内容

典型用法

  • 从一种类型转换为另一种类型
  • 处理可空性

apply

定义

  • 返回被调用的对象

典型用法

  • 初始化或配置对象

also

定义

  • 被调用对象通过参数传入
  • 返回被调用对象

典型用法

with

定义

  • 不是扩展功能
  • 相关对象作为参数传入
  • 返回lambda表达式返回的值

典型用法

  • 对对象进行逻辑分组调用

run

定义

  • 返回类型是lambda表达式返回的内容

典型用法
